A custom CMS that will host educational material about the Native American language Lakota.

<h3>Created with:</h3>

<h4>Frontend:</h4>
- React
- Redux
- TypeScript
- Bulma CSS

<h4>API/Backend:</h4>
- Elixir
- Phoenix
